# 🎮 Web-Based Games

A collection of fun browser-based games built with HTML, CSS, and vanilla JavaScript. Just open a file in your browser and play — no dependencies, no installs.

## Games

### 🐍 Snake

Classic snake game with a modern, polished look.

- **Smooth animations** with particle effects and trail rendering
- **Three difficulty levels** (Slow, Medium, Fast)
- **High score persistence** via localStorage
- **Mobile support** with on-screen d-pad controls
- **Pause/resume** with the spacebar
- **Keyboard controls:** Arrow keys or WASD to move, Space to pause, Enter to restart

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/sahitipchittam/web-based-games.git
   ```
2. Open `snake-game.html` in your browser.
3. Enjoy!
